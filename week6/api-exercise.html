Just like the Star Wars exercise, you're going to be developing a web page that displays data that is retrieved from some external api.

Suggestions

  • Pokemon API
  • Star Wars (don't use characters)
  • News API
  • Harry Potter API

Requirements

  1. Show using both the XMLHttpRequest as well as the fetch.
  2. Display at least 3 attributes on the page.
  3. If you're using the pokemon API or any API that includes pictures, try to get those pictures on the page. (Optional)
  4. Due Monday morning.
